# ArchiLog

An ASP.NET library that helps set up a fully functioning REST API.

### Features
- CRUD operations
- Searching
- Filtering
- Sorting
- Pagination
- Partial response
- Logs ( Serilog )
- Versionning
- Comments ( Swagger )
- Unit tests ( xUnit )
- Authentification ( JWT )
